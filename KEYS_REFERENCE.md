## Special

| Role | Key | Access |
|------|-----|--------|
| **Super Admin** | PhilFIDA-SuperAdmin | All regions, full read/write. Reusable — any number of accounts can be created with this key. |

---

## Usage

- **Admin key** — Create account → full access (add/edit/delete) for that region’s data.
- **Viewer key** — Create account → view-only access for that region’s data.
- **Super Admin key** — Create accounts with access to all regions; key can be reused.
